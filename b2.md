# BÀI 2: Xây dựng Thuật toán Khuyến mãi Phức tạp (Chain-of-thought - CoT)

## I. Phân tích Tầm quan trọng Thứ tự Ưu tiên Tính toán

**Vấn đề:**

- Tính tiền sai thứ tự → Tổn thất tài chính hoặc tính sai thuế
- Ví dụ: Nếu tính coupon trước khi trừ discount sản phẩm → coupon tính trên giá cao hơn
- Luật thuế: VAT tính trên tổng tiền sau chiết khấu (không tính trên phí vận chuyển)

**Tầm quan trọng:**

- Tuân thủ pháp luật về tính thuế
- Tránh tranh chấp với khách hàng (hiểu rõ quy tắc)
- Báo cáo tài chính chính xác
- Chain-of-thought giúp suy luận từng bước logic

## II. Prompt CoT Tối ưu

### **Persona**

Bạn là Senior Financial Architect của công ty fintech có kinh nghiệm tính toán giá tiền phức tạp trên các nền tảng thương mại điện tử.

### **Task**

Thiết kế thuật toán tính giá cuối cùng của đơn hàng theo quy tắc khuyến mãi phức tạp với độ chính xác tuyệt đối dùng BigDecimal.

### **Requirements - Chain-of-Thought**

Hãy áp dụng kỹ thuật CoT bằng cách:

1. **Bước 1: Xác định thứ tự ưu tiên đúng**
    - Liệt kê tất cả các thành phần tính tiền theo logic nghiệp vụ
    - Giải thích tại sao thứ tự này đúng (tham chiếu quy tắc thuế, quy tắc coupon)
    - Xác định input/output của mỗi bước

2. **Bước 2: Công thức chi tiết**
    - Viết công thức toán học cho từng bước
    - Chỉ định kiểu dữ liệu (BigDecimal với scale = 2)
    - Ghi rõ điều kiện logic (nếu... thì...)

3. **Bước 3: Dry-run bằng tay với ca kiểm thử cụ thể**
    - Input: Sản phẩm A (2x 400,000 VND/cái, giảm 10%), B (1x 300,000 VND, không giảm)
    - Coupon: -50,000 VND (tối thiểu 500,000 VND)
    - Khách: Gold member (giảm 5%)
    - Địa chỉ: 12km từ kho
    - Tính chi tiết từng bước, ghi tất cả các giá trị trung gian

4. **Bước 4: Tạo class Java OrderCalculator sử dụng BigDecimal**
    - Phương thức calculateTotal(Order order) → OrderPriceSummary
    - OrderPriceSummary chứa: originalAmount, productDiscount, couponDiscount, loyaltyDiscount, subtotal, vat, shippingFee, finalAmount

### **Output Format**

```
## Phân tích CoT
[Bước 1: Thứ tự ưu tiên]
[Bước 2: Công thức]
[Bước 3: Dry-run]
[Bước 4: Code Java]
```

## III. Response từ AI - Phân tích CoT Chi tiết

### **Bước 1: Xác định Thứ tự Ưu tiên Đúng**

```
1. Tính tổng tiền gốc (subtotal = sum(giá sản phẩm × số lượng))
2. Trừ Discount Sản phẩm (product discount)
3. Kiểm tra điều kiện coupon, trừ Coupon Code (nếu điều kiện met)
4. Trừ Loyalty Discount (tính trên subtotal sau coupon)
5. = Subtotal sau chiết khấu (base for VAT & shipping free)
6. Cộng VAT 10% (tính trên subtotal sau chiết khấu, KHÔNG tính vận chuyển)
7. Cộng Phí vận chuyển (hoặc miễn phí nếu > 1,000,000 VND)
8. = TOTAL FINAL
```

**Lý do:**

- Coupon tính SAU discount sản phẩm (tránh "double discount abuse")
- Loyalty discount tính TRƯỚC VAT (theo quy tắc VN)
- VAT KHÔNG tính trên phí vận chuyển (phí vận chuyển là dịch vụ, không hàng hóa)
- Miễn phí vận chuyển kiểm tra TRƯỚC khi cộng VAT

### **Bước 2: Công thức Chi tiết**

```
S1 = Σ(price_i × qty_i)                          // Tổng tiền gốc

S2 = S1 - Σ(discount_i)                          // Sau discount sản phẩm

S3 = S2                                           // Kiểm: S2 >= 500,000?
     - min(coupon_amount, 100,000) nếu S2 >= 500,000
     - 0 nếu S2 < 500,000

loyalty_pct = 0.05 (Gold) hoặc 0.10 (Platinum)
S4 = S3 - (S3 × loyalty_pct)

tax_amount = S4 × 0.10                           // VAT 10%

distance = 12 km
shipping = 30,000 + (distance - 10) × 5,000 nếu S4 < 1,000,000
         = 0 nếu S4 >= 1,000,000

TOTAL = S4 + tax_amount + shipping

Kiểu: BigDecimal (scale=2, RoundingMode.HALF_UP)
```

### **Bước 3: Dry-run Chi tiết**

**Input:**

- Sản phẩm A: 2 × 400,000 = 800,000 VND, giảm 10% → 800,000 - 80,000 = 720,000 VND
- Sản phẩm B: 1 × 300,000 = 300,000 VND, không giảm
- Coupon: -50,000 VND (tối thiểu 500,000 VND)
- Khách: Gold (giảm 5%)
- Địa chỉ: 12km

**Tính toán:**

```
S1 = 800,000 + 300,000 = 1,100,000 VND
S2 = 1,100,000 - 80,000 = 1,020,000 VND
S3 = 1,020,000 >= 500,000 ✓ → trừ 50,000 = 970,000 VND
S4 = 970,000 - (970,000 × 0.05) = 970,000 - 48,500 = 921,500 VND
Tax = 921,500 × 0.10 = 92,150 VND
Shipping = S4 = 921,500 < 1,000,000 → Tính: 30,000 + (12-10)×5,000 = 30,000 + 10,000 = 40,000 VND
TOTAL = 921,500 + 92,150 + 40,000 = 1,053,650 VND
```

**Breakdown:**

- Giá gốc: 1,100,000
- Giảm sản phẩm: -80,000
- Giảm coupon: -50,000
- Giảm loyalty: -48,500
- Subtotal trước thuế: 921,500
- VAT: 92,150
- Vận chuyển: 40,000
- **Tổng cuối: 1,053,650 VND**

### **Bước 4: Code Java OrderCalculator**

```java
@Getter
@AllArgsConstructor
public class OrderPriceSummary {
    private BigDecimal originalAmount;
    private BigDecimal productDiscount;
    private BigDecimal couponDiscount;
    private BigDecimal loyaltyDiscount;
    private BigDecimal subtotalAfterDiscounts;
    private BigDecimal vat;
    private BigDecimal shippingFee;
    private BigDecimal finalAmount;
}

@Service
@Slf4j
public class OrderCalculator {

    private static final BigDecimal VAT_RATE = new BigDecimal("0.10");
    private static final BigDecimal PRODUCT_DISCOUNT_RATE = new BigDecimal("0.10");
    private static final BigDecimal GOLD_LOYALTY_RATE = new BigDecimal("0.05");
    private static final BigDecimal PLATINUM_LOYALTY_RATE = new BigDecimal("0.10");
    private static final BigDecimal COUPON_MAX_AMOUNT = new BigDecimal("100000");
    private static final BigDecimal COUPON_MIN_SUBTOTAL = new BigDecimal("500000");
    private static final BigDecimal SHIPPING_MIN_FREE = new BigDecimal("1000000");
    private static final BigDecimal SHIPPING_BASE = new BigDecimal("30000");
    private static final BigDecimal SHIPPING_PER_KM = new BigDecimal("5000");
    private static final BigDecimal KM_THRESHOLD = new BigDecimal("10");

    public OrderPriceSummary calculateTotal(Order order) {
        log.info("=== START ORDER CALCULATION ===");

        // Step 1: Tính tổng tiền gốc
        BigDecimal s1 = calculateOriginalAmount(order.getItems());
        log.info("S1 (Original Amount): {}", s1);

        // Step 2: Trừ discount sản phẩm
        BigDecimal productDiscount = s1.multiply(PRODUCT_DISCOUNT_RATE)
            .setScale(2, RoundingMode.HALF_UP);
        BigDecimal s2 = s1.subtract(productDiscount);
        log.info("S2 (After Product Discount): {} (discount: {})", s2, productDiscount);

        // Step 3: Trừ coupon
        BigDecimal couponDiscount = calculateCouponDiscount(s2, order.getCouponCode());
        BigDecimal s3 = s2.subtract(couponDiscount);
        log.info("S3 (After Coupon): {} (coupon: {})", s3, couponDiscount);

        // Step 4: Trừ loyalty discount
        BigDecimal loyaltyDiscount = calculateLoyaltyDiscount(s3, order.getCustomer().getMembershipTier());
        BigDecimal s4 = s3.subtract(loyaltyDiscount);
        log.info("S4 (After Loyalty Discount): {} (loyalty: {})", s4, loyaltyDiscount);

        // Step 5: Cộng VAT (tính trên S4)
        BigDecimal vat = s4.multiply(VAT_RATE)
            .setScale(2, RoundingMode.HALF_UP);
        log.info("VAT (10% of S4): {}", vat);

        // Step 6: Cộng phí vận chuyển
        BigDecimal shippingFee = calculateShippingFee(s4, order.getShippingDistance());
        log.info("Shipping Fee: {}", shippingFee);

        // Step 7: Tính tổng cuối
        BigDecimal finalAmount = s4.add(vat).add(shippingFee)
            .setScale(2, RoundingMode.HALF_UP);

        log.info("=== FINAL TOTAL: {} ===", finalAmount);

        return new OrderPriceSummary(
            s1,
            productDiscount,
            couponDiscount,
            loyaltyDiscount,
            s4,
            vat,
            shippingFee,
            finalAmount
        );
    }

    private BigDecimal calculateOriginalAmount(List<OrderItem> items) {
        return items.stream()
            .map(item -> item.getPrice()
                .multiply(new BigDecimal(item.getQuantity()))
                .setScale(2, RoundingMode.HALF_UP))
            .reduce(BigDecimal.ZERO, BigDecimal::add);
    }

    private BigDecimal calculateCouponDiscount(BigDecimal subtotal, String couponCode) {
        if (couponCode == null || couponCode.isBlank()) {
            return BigDecimal.ZERO;
        }

        // Kiểm điều kiện: subtotal >= 500,000?
        if (subtotal.compareTo(COUPON_MIN_SUBTOTAL) < 0) {
            log.warn("Coupon not applied: subtotal {} < minimum {}", subtotal, COUPON_MIN_SUBTOTAL);
            return BigDecimal.ZERO;
        }

        // Coupon giảm 50,000 nhưng tối đa 100,000
        BigDecimal couponAmount = new BigDecimal("50000");
        return couponAmount.min(COUPON_MAX_AMOUNT);
    }

    private BigDecimal calculateLoyaltyDiscount(BigDecimal subtotal, String membershipTier) {
        BigDecimal rate = "PLATINUM".equals(membershipTier)
            ? PLATINUM_LOYALTY_RATE
            : "GOLD".equals(membershipTier)
            ? GOLD_LOYALTY_RATE
            : BigDecimal.ZERO;

        return subtotal.multiply(rate)
            .setScale(2, RoundingMode.HALF_UP);
    }

    private BigDecimal calculateShippingFee(BigDecimal subtotal, int distanceKm) {
        // Nếu subtotal >= 1,000,000 → miễn phí
        if (subtotal.compareTo(SHIPPING_MIN_FREE) >= 0) {
            log.info("Shipping fee waived (subtotal >= 1,000,000)");
            return BigDecimal.ZERO;
        }

        // Tính: 30,000 + (distance - 10) * 5,000 nếu distance > 10km
        if (distanceKm <= 10) {
            return SHIPPING_BASE;
        }

        BigDecimal extraKm = new BigDecimal(distanceKm).subtract(KM_THRESHOLD);
        return SHIPPING_BASE.add(extraKm.multiply(SHIPPING_PER_KM))
            .setScale(2, RoundingMode.HALF_UP);
    }
}
```
